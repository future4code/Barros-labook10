<h2>O LaBook será uma rede social com o objetivo de promover a conexão e interação entre seus mais diversos usuários. </h2>

<h2>Endpoints: </h2>


<h3>Usuários</h3>
✅ Pegar todos os usuários
✅ Criar usuário

<h3>Posts</h3>
✅ Criar Post
✅ Pegar Post pelo ID
✅ Feed dos posts dos amigos

<h3>Amizades</h3>
✅ Fazer amizade
✅ Desfazer amizade
✅ Pegar todas as amizades
